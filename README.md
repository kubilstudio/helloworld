# 🚢 Hello World Ship Animation

A simple Python terminal animation project that displays a colorful **"Hello World" ship journey** using ANSI colors and ASCII art.

This project creates a small ocean scene with a ship, welcome message, and wave animation in the terminal.

## ✨ Features

- 🚢 ASCII ship animation
- 🌊 Ocean wave animation
- 🎨 ANSI terminal colors
- ⏱️ Simple time-based animation
- 💻 Works on Windows / Linux / macOS terminal

## 🛠️ Requirements

## 📥 Installation

Clone this repository:

```bash
git clone https://github.com/your-username/hello-world-ship.git
```

```bash
cd hello-world-ship
```

Run the Python file:

```bash

```

You will see the ship animation start in your terminal 🚢

## 🧩 How It Works

The program uses:

- `time.sleep()`  
  for animation delays

- `os.system()`  
  to clear the terminal screen

- ANSI escape codes  
  to display colored text

- ASCII Art  
  to draw the ship
